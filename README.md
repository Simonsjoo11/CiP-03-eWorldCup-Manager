## How to run the project

### Prerequisites

- .NET SDK 9.0+
- Node.js 18+ (LTS) and npm

---

### 1) Start the backend (ASP.NET Core)

```bash
cd backend/EWorldCup.Api
dotnet restore
dotnet run
```

---

### 2) Start the frontend (Next.js)

```bash
cd frontend
npm install
npm run dev
```
